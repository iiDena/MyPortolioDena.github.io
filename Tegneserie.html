Comic Con Website

Project Description

You have to re-design a comic con website in order to improve the site's accessibility. You must change all parts of the website (code, images, visual design etc.), as long as you still convey the site's current information. In your work, you have to deal with as many of the areas within accessibility as possible. Your design proposal must be coded in HTML/CSS and uploaded to the web hotel.

Details

  • Topic: Accessibility
  • Softwares/tools: Illustrator, Figma, In Design, Visual Studio, html, css, Javascript
  • Project Duration: 5 days
  • Project Website: Comic Con Website

Accessibility in our website

  • The site's content, language and presentation
  • Semantics
  • Visual design
  • Code structure
  • Performance
  • Keyboard
