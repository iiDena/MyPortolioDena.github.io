Oats Landingpage

Project Description

On landingpage, there must be a description of the selected company, as well as a description of the selected product or service. Landingpage must be built with Bootstrap and it must have a layout that automatically adapts to different screen sizes (responsive design). You need to investigate the selected company and examine the target audience

This project was about finding a product to brand, and what better product to brand than oats? Since a lot of people find healthy food boring we wanted to make it look more tempting to buy buy and specifically we wanted to brand it to kids.

Details

  • Topic: Branding
  • Softwares/tools: Illustrator, XD, Visual studio, HTML, CSS, bootstrap
  • Project date: December - 2021
