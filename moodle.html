School Website (Moodle)

Project Description

This project was split in 3 parts all working towards researching and creating a better prototype for our school website. First part was researching which was based mostly by user testing the current website with oberving students using it and interviewing them. Afterwards creating empathy maps and personas

Second part was based of all our results from the first part, so working forward using our persona. We had to figure out how to make the website better and which new features we should add. We worked with User Stories , "How might we?" questions and from there created wireframes.

Final part was about creating testing our wireframes in-person and remote. We thereafter looked at some Design principles and ways to simplify our website with a final prototype.

Details

  • Topic: Research, User Cases, HMW, Usertests & simplifying
  • Softwares/tools: Figma
  • Project Duration: 3 weeks
  • Project Website: Figma Prototype
